The easiest way to stay in touch with your customers 24×7.

== Description ==

The floating Button for WhatsApp for websites developed by Autochat.uy allows you to embed it natively on your site, so that the user naturally makes his or her questions through this popular messaging platform.

More and better leads.
Increases leads with existing traffic, since customers are encouraged to ask questions directly using Whatsapp.
The button also automatically, marks an unread message after a few seconds to the visitor, so that he or her is encouraged to check the message and respond.

Completely customizable tool for any language.
It is a completely customizable tool for any language, which allows you to add name, the photo you want, the state you are in (for example: online, offline, busy, available) and the initial message to start the chat with the client. It also allows you to select the position where we want the button to be displayed on the website, left or right in the bottom of your website.

Mobile adaptative.
The Button for WhatsApp for websites developed by Autochat.uy allows you to embed it natively on your site, so that the user naturally makes his or her questions through this popular messaging platform.

Friendly installation.
You do not need to know how to program. By installing the plugin on your Wordpress site and configuring the message and your phone, you will start receiving leads about your products and services.

By default, the language in which the plugin options are displayed will be the language of your wordpress site panel. Anyway, if you complete the data fields manually you can use the texts you want, since they are completely customizable.

== Installation ==

Follow this steps:
 - Download the plugin.
 - Open WordPress admin, go to Plugins, click Add New. Then upload Plugin
 - Click select file button and upload the .zip file.
 - Activate and open plugin’s settings page located in the main WP admin menu as “Auy Whatsapp”.
 
Or if needed, upload manually:
 - Download the plugin.
 - Unzip it and upload to /wp-content/plugins/
 - Open WordPress admin – Plugins and click “Activate” next to the plugin
 - Activate and open plugin’s settings page located in the main WP admin menu as “Auy Whatsapp”.

The plugin suports some translations. By default labels and texts 
Check out our [documentation](https://autochat.uy/readme-whatsapp/) to make the most of our Button for WhatsApp plugin.
